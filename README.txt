SNAKE AI iOS PROJECT

HOW TO USE:

1. Open Xcode
2. Create new project:
   - iOS > App
   - Interface: UIKit
   - Language: Swift
   - Disable Core Data

3. Replace files with these:
   - AppDelegate.swift
   - GameViewController.swift
   - GameScene.swift

4. Run on simulator or iPhone

NOTE:
- This is offline native SpriteKit game
- Hamiltonian AI movement included (simple version)
